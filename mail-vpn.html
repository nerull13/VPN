------------------------------------------------
Pour Configurer votre VPN
------------------------------------------------

Nous vous invitons a lire le PDF suivant, avec photos, pour plus de details : Tutoriel VPN

1.Pour telecharger le module OpenVPN pour Windows, visitez la page de telechargements OpenVPN :

OpenVPN

2.Selectionnez le logiciel en fonction de votre Windows :

3.Exécutez le programme d'installation

4.Selectionnez votre dossier de destination et cliquez sur Installer.

5.L’assistant vous previendra lorsque l’installation sera terminee et vous demandera de cliquer sur Terminer.

(un redémarrage peut être nécessaire)

6.Téléchargez le fichier de configuration OpenVPN en pièce jointe de l'email ( .ovpn )

7. une fois le logiciel lancé, faites un clique droit sur l'icône du logiciel en forme d'écran à coté de l'heure et faire "importer fichier ..." > puis selectionner le fichier OVPN

 

---------------------------------
Pour les MAC
---------------------------------
Vous devez suivre ces étapes :

Télécharger et installer Tunnelblick. à partir de : https://tunnelblick.net
Telecharger le fichier de configuration .opvn joint à cet email
Ouvrer le fichier joint à cet email, cela ouvrira Tunnelbrick


------------------------------------------------
Pour se connecter et déconnecter
------------------------------------------------

Vérifiez toujours que STUNNEL est démarré avant de lancer OpenVPN si vous choisissez le profil avec contournement du pare-feu

1.Trouvez l'icone OpenVPN sur votre bureau et démarrer le logiciel

2. Faite un clique droit sur l'icône du logiciel dans la barre de tâches

3.Trouvez le serveur auquel vous voulez vous connecter dans le menu, et choisissez connecter/déconnecter dans le sous-menu du site.

4.L'icone de OpenVPN GUI dans le plateau deviendra verte une fois la connexion securisée établie.

5.Verifiez votre adresse IP sur : Vérifier mon IP avec Ip Tracker
